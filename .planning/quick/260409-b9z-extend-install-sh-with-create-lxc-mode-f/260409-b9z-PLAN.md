Add a `do_create_lxc()` function to install.sh between the `do_uninstall` function and the main command parser. This function:

1. **Parse flags** before anything else. Loop through "$@" to extract:
   - `--storage VALUE` (default: `local-lvm`) for rootfs storage
   - `--bridge VALUE` (default: `vmbr0`) for network bridge
   - `--hostname VALUE` (default: `charging-master`) for container hostname

2. **Proxmox check**: Verify `pct` command exists. If not, error "This command must be run on a Proxmox host." and exit 1.

3. **Root check**: Same pattern as other modes — `id -u` must be 0.

4. **Template handling**:
   - Run `pveam list local` and grep for `debian-13`. Capture the template filename.
   - If no match found, log "Downloading Debian 13 template..." and run `pveam download local debian-13-standard_13.0-1_amd64.tar.zst`. Then re-check `pveam list local` for the exact filename.
   - Store the template filename in a variable (e.g., `TEMPLATE`). The template reference for pct create is `local:vztmpl/${TEMPLATE}`.

5. **Get next VMID**: `pvesh get /cluster/nextid` — store in `VMID` variable.

6. **Create container**:
   ```
   pct create "$VMID" "local:vztmpl/${TEMPLATE}" \
     --hostname "$CT_HOSTNAME" \
     --cores 1 \
     --memory 1024 \
     --swap 512 \
     --rootfs "${STORAGE}:4" \
     --net0 "name=eth0,bridge=${BRIDGE},ip=dhcp" \
     --unprivileged 1 \
     --features nesting=1 \
     --start 1
   ```
   Note: `nesting=1` is needed so systemd works properly inside the container.

7. **Wait for network**: Sleep 5 seconds, then poll up to 30 seconds for an IP:
   ```bash
   local ip=""
   local retries=0
   while [ -z "$ip" ] && [ "$retries" -lt 6 ]; do
     sleep 5
     ip=$(pct exec "$VMID" -- hostname -I 2>/dev/null | awk '{print $1}') || true
     retries=$((retries + 1))
   done
   ```
   If still no IP after retries, warn but continue.

8. **Bootstrap inside container**:
   ```bash
   log "Installing charging-master inside CT ${VMID}..."
   pct exec "$VMID" -- bash -c "apt-get update -qq && apt-get install -y -qq curl git > /dev/null"
   pct exec "$VMID" -- bash -c "curl -sSL https://raw.githubusercontent.com/meintechblog/charging-master/main/install.sh | bash -s -- install"
   ```

9. **Print result**:
   ```
   log "LXC container created successfully!"
   log "  VMID:     ${VMID}"
   log "  Hostname: ${CT_HOSTNAME}"
   log "  IP:       ${ip:-unknown}"
   log ""
   if [ -n "$ip" ]; then
     log "Charging-Master is running at: http://${ip}:3000"
   else
     log "Could not detect IP. Check: pct exec ${VMID} -- hostname -I"
   fi
   ```

Also update the main case statement to add `create-lxc` and update the usage text:
```
  create-lxc) do_create_lxc "$@" ;;
```
